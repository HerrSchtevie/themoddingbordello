## 🔢 Versioning & Save Safety

This modlist uses a three-part semantic versioning format: `MAJOR.MINOR.PATCH`.

Save compatibility depends on **which number changes** between releases.

- **❌ DEFINITELY NOT save-safe — MAJOR version change (first number)**  
  Examples: `4.2.0 → 5.0.0`, `5.1.3 → 6.0.0`  
  Major rebuilds or fundamental changes. **Always requires a new game.**

- **❌ NOT save-safe — MINOR version change (second number)**  
  Examples: `4.2.0 → 4.3.0`, `5.1.0 → 5.2.0`  
  Structural or gameplay changes. **Requires a new game.**

- **✅ Save-safe — PATCH version change (third number)**  
  Examples: `4.2.0 → 4.2.1`, `5.1.0 → 5.1.1`  
  Bug fixes, tuning, or minor adjustments. **Safe to update mid-save.**
