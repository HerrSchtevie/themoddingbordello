## 📦 Manual Downloads – Mantras of Mara (MOM)

This page exists to document **all files that must be downloaded manually** for **Mantras of Mara (MOM)** in order for the Wabbajack installation to complete successfully.

Due to hosting restrictions, permissions, or site limitations (most commonly on LoversLab and select third-party pages), **some required mods cannot be downloaded automatically by Wabbajack**. These files must be downloaded by hand and placed into your MOM downloads folder before or during installation.

This is **normal behavior** for modlists that utilize LoversLab mods and is **not an error** with MOM, Wabbajack, or your setup.

### How This Page Is Used
- Every file listed here is **required** for MOM.
- If Wabbajack encounters a missing file, it will tell you **exactly which file is missing**.
- Simply locate that file on this page, download it manually, and place it into your MOM downloads directory.
- Once all required files are present, Wabbajack will proceed normally.

### Important Notes
- **Double-check file versions and names.** Incorrect versions will still fail.
- **Do not rename files.**
- Manual downloads are a prerequisite before requesting support for install errors.

This page is kept up to date to ensure a **smooth, repeatable installation experience** for all MOM users.
